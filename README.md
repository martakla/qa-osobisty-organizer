# Dokumentacja testowa — OsobistyOrganizer

Repozytorium zawiera dokumentację testową przygotowaną dla aplikacji **OsobistyOrganizer — Todo App**.

## Testowana aplikacja

**Demo aplikacji:** https://osobisty-organizer-umvv.vercel.app/  
**Repozytorium aplikacji:** https://github.com/martakla/OsobistyOrganizer

## Cel projektu

Celem projektu jest przygotowanie podstawowej dokumentacji QA dla aplikacji webowej do zarządzania zadaniami oraz przećwiczenie procesu testowania manualnego.

## Zakres dokumentacji

Dokumentacja obejmuje:

- checklistę testową,
- przypadki testowe z danymi testowymi, statusami i powiązaniem z przypadkami użycia,
- zgłoszenia błędów i obserwacje,
- przypadki użycia,
- diagram przypadków użycia.

## Testowane obszary

- rejestracja użytkownika,
- logowanie użytkownika,
- dodawanie zadań,
- przypisywanie kategorii,
- ustawianie terminów,
- filtrowanie zadań,
- sortowanie zadań według terminu,
- oznaczanie zadań jako wykonane,
- usuwanie zadań,
- podstawowa obsługa dostępu użytkownika.

## Zawartość repozytorium

- `checklist.md` — checklista testowa aplikacji
- `test-cases.md` — przypadki testowe z danymi testowymi, statusami i powiązaniem z przypadkami użycia
- `bug-reports.md` — zgłoszenia błędów i obserwacje
- `use-cases.md` — opis przypadków użycia aplikacji
- `use-cases-diagram.md` — diagram przypadków użycia
- `assets/use-cases-diagram.png` — plik graficzny z diagramem przypadków użycia
