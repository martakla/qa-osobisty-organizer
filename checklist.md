# Checklist testowa — Osobisty Organizer

## Rejestracja i logowanie

- [ ] Użytkownik może zarejestrować konto poprawnymi danymi
- [ ] Użytkownik może zalogować się poprawnymi danymi
- [ ] Aplikacja obsługuje błędne dane logowania
- [ ] Formularze nie powinny akceptować pustych wymaganych pól
- [ ] Po zalogowaniu użytkownik widzi stronę główną aplikacji

## Zadania

- [ ] Użytkownik może dodać nowe zadanie
- [ ] Użytkownik może usunąć zadanie
- [ ] Użytkownik może oznaczyć zadanie jako wykonane
- [ ] Użytkownik może przypisać kategorię do zadania
- [ ] Użytkownik może ustawić termin wykonania zadania
- [ ] Zadanie po dodaniu pojawia się na liście

## Filtrowanie

- [ ] Użytkownik może filtrować zadania według kategorii
- [ ] Po zmianie filtra lista zadań aktualizuje się poprawnie
- [ ] Po wybraniu kategorii widoczne są tylko pasujące zadania

## Terminy

- [ ] Aplikacja wyświetla czas pozostały do terminu zadania
- [ ] Aplikacja poprawnie obsługuje zadania z przyszłym terminem
- [ ] Aplikacja poprawnie obsługuje zadania po terminie

## Responsywność

- [ ] Aplikacja działa poprawnie na desktopie
- [ ] Przyciski są widoczne i łatwe do kliknięcia
- [ ] Lista zadań jest czytelna
- [ ] Komunikaty i informacje dla użytkownika są zrozumiałe
